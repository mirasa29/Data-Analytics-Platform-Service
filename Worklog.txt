# Starting with Docker compose file to setup the environment

- Issue with loading data on install via docker container
cause:
    - wrong timestamp format in the data (1/1/2023 8:09 AM)
    - unicode settings UTF-8 doesn't support currency format '£'
workaround:
    - manual data manipulation by changing the timestamp format to '2023-01-01 08:09:00'
    - removed '£'
    - ran docker
    - ran `docker exec -i sales_postgres_db psql -U sales_user -d sales_db -f /docker-entrypoint-initdb.d/init.sql`
